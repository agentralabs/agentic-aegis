# Installation

## npm (Recommended)

```bash
npm install @agenticamem/aegis
```

## Cargo

```bash
cargo install agentic-aegis
```

## From Source

```bash
git clone https://github.com/agentra/agentic-aegis
cd agentic-aegis
cargo build --release
```

The binaries `aegis` and `agentic-aegis-mcp` will be in `target/release/`.

## Python Bindings

```bash
pip install agentic-aegis
```

## Verify Installation

```bash
aegis --version
```
