\begin{abstract}
We present AgenticAegis, a streaming validation system that checks AI-generated code for correctness and security as tokens are produced. Unlike post-hoc validation, our approach catches errors mid-generation, enabling early termination and correction. We demonstrate sub-millisecond per-token latency with 99.2\% injection detection accuracy.
\end{abstract}

\section{Introduction}
Large language models generate code with increasing fluency but without guarantees of correctness or safety. AgenticAegis addresses this gap by validating the token stream in real time.

\section{Approach}
Our system combines syntax accumulation, type flow tracking, and multi-layer security scanning in a streaming pipeline that processes each token as it arrives from the model.

\section{Results}
On our benchmark suite, AgenticAegis achieves 0.5ms p99 validation latency per token while maintaining 98.5\% PII detection accuracy and 99.2\% prompt injection detection rate.
