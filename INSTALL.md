# Install AgenticAegis

## npm

```bash
npm install @agenticamem/aegis
```

## Cargo

```bash
cargo install agentic-aegis
```

## From Source

```bash
git clone https://github.com/agentra/agentic-aegis
cd agentic-aegis
cargo build --release
```

## Python

```bash
pip install agentic-aegis
```

## Verify

```bash
aegis --version
agentic-aegis-mcp --version
```
